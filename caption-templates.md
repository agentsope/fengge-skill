发图集时直接复制粘贴用。每套都带钩子 + 内容 + tag。

## 发布建议

| 场景 | 推荐套餐 | 图片用哪张 |
|---|---|---|
| 第一次发，引爆话题 | A 或 B | 总封面 + 4 张矩阵 |
| 想引流到 GitHub | C | 总封面 |
| 单图测试效果 | D | 选一张最有梗的(压抑峰或二次元峰) |
| 互动型 | E | 4 宫格汇总图 |
| 抖音视频封面 | (不用 caption) | 总封面 |
| 朋友圈 | A 或 D | 4 宫格汇总图 |
| 微博长贴 | B | 5 张全发 |

## 共用 hashtag 池(从中挑 5-8 个)

- 必带:`#SkillAlchemy` `#峰哥亡命天涯`
- 主题:`#AI蒸馏` `#Persona` `#蒸馏版峰哥`
- 玩梗:`#资本峰回头是岸` `#酒鬼峰真情流露` `#压抑峰底层智慧` `#胡子大叔说jksl`
- 圈子:`#B站二圣` `#ClaudeCode` `#AI玩梗`
- 万能:`#AIGC` `#人工智能`
